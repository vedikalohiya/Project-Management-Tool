# Project Management Tool

A full-stack Project Management Tool built to help teams efficiently manage projects, tasks, deadlines, and team collaboration in one centralized platform. This application improves workflow management by providing task tracking, project monitoring, user authentication, and real-time team productivity features.

## 🚀 Features

- User Authentication & Authorization
- Create, Update, Delete Projects
- Task Assignment & Progress Tracking
- Team Collaboration Dashboard
- Deadline & Priority Management
- Project Status Monitoring
- Responsive UI for all devices
- REST API Integration
- Role-Based Access Control
- Clean and Scalable Architecture
