# 🚀 Project Management Tool

A modern Kanban-based Project Management Application built using React, Vite, and Supabase to help teams manage tasks, workflows, and collaboration efficiently.

---

## ✨ Features

- 📋 Kanban-style task management
- 📝 Create, update, and delete tasks
- 📌 Drag-and-drop workflow organization
- 🔐 Secure authentication with Supabase
- ⚡ Fast and responsive UI with React + Vite
- ☁️ Cloud database integration
- 📱 Responsive design for desktop and mobile

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- Supabase
- HTML5 & CSS3

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/vedikalohiya/Project-Management-Tool.git
```

### 2️⃣ Navigate to the Project

```bash
cd Project-Management-Tool
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Configure Environment Variables

Create a `.env` file inside `kanban-app/` and add:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 5️⃣ Run the Development Server

```bash
npm run dev
```

The application will start at:

```bash
http://localhost:5175
```
